How Fugu is my browser?

The cross-company effort Project Fugu has the objective of making it possible for Web apps to do anything iOS, Android, or desktop apps can.

We do this by exposing the capabilities of these platforms to the Web, all while maintaining user security, privacy, trust, and other core tenets of the Web.
